🔍 Recommendation Mode

  • VPA observes real CPU/memory usage
  • Goldilocks creates one VPA per Deployment
  • Controller reads recommendations + current requests
  • Calculates cost delta (AWS on-demand pricing)
  • Posts to #k8s-hack-resizer Slack channel

⚡ Auto-Resize Mode

  • Off by default — explicit opt-in required
  • Threshold: only act when deviation > 20%
  • Patches resources.requests only (never limits)
  • Runs on a configurable schedule (60 min)
  • Toggle with /resize autoapply on

🔒 Safety Constraints

  • RBAC scoped to only prajwal-hack + prajwal-hack-region-1
  • VPA in recommendation-only mode (no auto-eviction)
  • Escape hatch: hack-resizer/skip: "true" annotation
  • No other cluster components touched

💬 Slash Commands

/resize recommend → show all savings /resize apply → patch all now /resize status → current state /resize autoapply on|off
